# 🐶🐱 Cat vs Dog Classifier (Transfer Learning)

This project is an AI web app that predicts whether an uploaded image is a **Cat** or a **Dog** using Transfer Learning.

The frontend is built using Streamlit and the model is trained using TensorFlow.

## 📌 Features
✔ Upload image  
✔ Predict Cat or Dog  
✔ Confidence score  
✔ Clean Streamlit UI  
✔ Transfer Learning using MobileNetV2  

---

## 🧠 Model Details
- Pretrained Model: MobileNetV2  
- Framework: TensorFlow  
- Technique: Transfer Learning  
- Dataset: Kaggle Cat vs Dog dataset
